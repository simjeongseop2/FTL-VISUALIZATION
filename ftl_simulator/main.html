FTL Visualization

FTL의 visualization

해당 Web application은 FTL(Flash Translation Layer)의 동작 과정을 애니메이션 형식으로 보여줌으로써 해당 FTL에 대해 처음 배우는 학생들의 이해를 돕기 위한 목적으로 제작되었습니다.
FTL 알고리즘은 SECTOR Based FTL, PAGE Based FTL, BLOCK Based FTL, HYBRID MAPPING FTL, DFTL 이렇게 총 5가지로 구성되어있습니다.
또한 각 알고리즘 별 애니메이션 속도 조절, 부분 재생, 실행 취소, 파일 불러오기 등 이해를 돕기 위한 부수적인 기능들을 제공하고 있습니다.

File 작성 방법

text File에 다음의 방식으로 Operation을 미리 적어 놓고, 그것을 불러와서 실행할 수 있습니다.
Write Operation은 (write 'lpn' 'data') 순으로 적습니다. ex) write 1 2 (1번 lpn에 data 2를 write하는 operation 실행)
Read Operation은 (read 'lpn') 순으로 적습니다. ex) read 1 (1번 lpn을 read하는 operation 실행)
연속된 lpn 혹은 data는 '-'을 이용해 표현할 수 있습니다. ex) write 1-10 1-10 (1번 lpn부터 10번 lpn에 각각 1부터 10까지의 data를 write하는 operation 실행)
같은 Operation에 대해서 여러 lpn 혹은 data를 한 줄에 표현할 수 있습니다. ex) read 1,3,5 (1번, 3번, 5번 lpn을 read하는 operation 실행)
